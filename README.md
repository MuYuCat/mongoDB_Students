    2.  创建网站服务器实现客户端和服务器端通信

#### 流程
    1.  建立项目文件夹并生成项目描述文件
    2.  创建网站服务器实现客户端和服务器端通信
    3.  连接数据库并根据需求设计学员信息表
    4， 创建路由并实现页面模板呈递
    5.  实现静态资源访问
    6.  实现学生信息添加功能
    7.  实现学生信息展示功能
#### 添加学生信息功能步骤分析
    1.  在模板的表单中指定请求地址与请求方式
    2.  为每一个表单项添加name属性
    3.  添加实现学生信息功能路由
    4.  接收客户端传递过来的学生信息
    5.  将学生信息添加到数据库中
    6.  将页面重定向到学生信息列表页面
#### 学生信息列表页面分析
    1.  从数据库中将所有的学生信息查询出来
    2.  通过模板引擎将学生信息和HTML模板进行拼接
    3.  将拼接好的HTML模板响应给客户端
